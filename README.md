# HabitFlow – Social Habit Tracking Platform

HabitFlow is a MEAN Stack web application where users can track their daily habits and connect with people who share similar goals like coding, fitness, reading, or studying.

## Tech Stack
MongoDB  
Express.js  
Angular  
Node.js  

## Features
- Habit tracking
- Habit streak system
- User discovery with similar interests
- Social interaction and community feed
- Dashboard analytics

## Project Structure
backend/ – Node.js + Express APIs  
frontend/ – Angular user interface
